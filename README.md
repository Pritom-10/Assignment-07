# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a modern friendship management web app built with React.  
It helps users track, manage, and maintain meaningful connections through smart reminders, interaction history, and analytics.

---

## 🚀 Live Demo
https://your-live-link.com

---

## 📌 Features

### 🧑 Friends Management
- Display friends from JSON data
- Beautiful responsive friend cards
- Click any friend to view full details

### 👤 Friend Details Page
- Profile overview with tags, bio, and status
- Quick actions: Call, Text, Video
- Action buttons generate timeline entries

### ⚡ Timeline System
- Tracks all interactions (Call / Text / Video)
- Filter timeline by interaction type
- Shows date, icon, and title for each entry

### 📊 Analytics Dashboard
- Pie chart using Recharts
- Visual breakdown of interaction types
- Real-time data from user activity

### 🔔 Toast Notifications
- Instant feedback on actions
- Custom styled notifications for Call, Text, Video

### 📱 Responsive Design
- Fully responsive for mobile, tablet, and desktop
- Clean UI following modern design patterns

### 🧭 Navigation
- Navbar with active route highlight
- Smooth routing using React Router

### ❌ 404 Page
- Custom not found page for invalid routes
